# 🤖 AI-Powered College Attendance Management System

## 📌 Project Overview

The **AI-Powered College Attendance Management System** is an intelligent web application designed to monitor, analyze, and improve student attendance using smart logic and automation.

This system not only tracks attendance but also helps students stay consistent by sending alerts and motivational messages based on their attendance performance.

---

## 🚀 Features

### 🔐 Authentication System

* User Signup & Login
* Input validation (email & password)
* Secure user data handling (LocalStorage / Firebase / Backend)

### 📊 Smart Dashboard

* Displays:

  * Total Classes
  * Attended Classes
  * Attendance Percentage
* Dynamic charts and progress bars

### ⚠️ Attendance Alert System

* Automatically detects attendance below **70%**
* Shows warning notification:

  > "Warning: Your attendance is below 70%"
* Detailed report view option

### 💬 Motivation System

* Sends motivational messages for **100% attendance**
* Dynamic message generator
* Encourages discipline and consistency

### 📈 Attendance Management

* Mark attendance (Present / Absent)
* Real-time percentage calculation
* Data stored and updated instantly

### 🔔 Notification System

* Notification bell with dropdown
* Alerts and messages display
* Mark as read feature

### 📩 Contact Form

* User input validation
* Stores or processes messages
* Success / error feedback

### 🌙 Dark Mode

* Toggle between Light and Dark themes
* Saves user preference

## ⚙️ How It Works

1. User logs into the system
2. Attendance is recorded daily
3. System calculates attendance percentage
4. If attendance < 70% → Alert triggered
5. If attendance = 100% → Motivation message shown
6. Dashboard updates in real-time
